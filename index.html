// 01 — Data Engineering · Cargo Analytics

CARGO
COMMODITY
STANDARDIZATION

20,000+ cargo records across three source systems carried inconsistent, multilingual descriptions filled with abbreviations, input errors, and formatting chaos — making classification, customs reporting, and analytics impossible at scale.

Problem

Cargo descriptions in operational systems were unstructured: mixed FR/EN/AR languages, abbreviations, typos, and free-text entries. Downstream analysis — commodity classification, customs reporting, volume tracking — was unreliable and took days manually.

Approach
  • Analysed raw description patterns across 3 systems
  • Python NLP pipeline: lowercasing, diacritics, stopwords
  • Regex patterns for abbreviations, units, codes
  • Fuzzy matching against master taxonomy (85% threshold)
  • Keyword-based rules engine for edge cases
  • Power Query integration for Excel outputs
Architecture

Python ETL pipeline ingests raw cargo manifests → applies multi-stage text cleaning → maps to master commodity taxonomy → exports to Excel for operational use and Power Query for downstream reporting. Fully repeatable for future batches.

// 02 — Business Intelligence · Dashboarding

COMMERCIAL
PERFORMANCE
DASHBOARD

Commercial and operational metrics were scattered across dozens of standalone spreadsheets. Leadership had no real-time consolidated view of container volumes, client activity, or revenue performance.

Problem

KPIs lived in silos: container volumes in one file, client data in another, vessel schedules elsewhere. Producing a combined performance view took hours of manual assembly each week — introducing lag and errors into commercial decisions.

Approach
  • Mapped all data sources and KPI requirements
  • Designed centralised Power BI star schema model
  • Built DAX measures: volume, revenue, activity
  • Created drill-through pages by client, vessel, period
  • Connected Excel via Power Query transformations
Architecture

Excel sources → Power Query (ETL) → Power BI Star Schema → DAX calculated measures → Interactive dashboards with executive overview, client-level, vessel/voyage breakdowns — all filterable by time, segment, and geography.
